Animation Parameter Examples

3d animation

Animations with the 3D settings will attempt to generate frames with depth, as the canvas moves according to the 3D specific parameters.

Translation_X

Moves the canvas left or right. This parameter uses positive values to move right and negative values to move left.


Positive

Negative

Translation_Y

Moves the canvas up or down. This parameter uses positive values to move up and negative values to move down.


Positive

Negative

Translation_Z

Moves the canvas towards or away from view. This parameter uses positive values to move forwards and negative values to move backwards.


Positive

Negative

Rotation_X

Tilts the canvas up or down in degrees per frame. This parameter uses positive values to tilt up and negative values to tilt down.


Positive

Negative

Rotation_Y

Pans the canvas left or right in degrees per frame. This parameter uses positive values to pan right and negative values to pan left.


Positive

Negative

Rotation_Z

Rolls the canvas clockwise or counterclockwise in degrees per frame. This parameter uses positive values to roll counterclockwise and negative values to roll clockwise.


Positive

Negative

2d animation

Animations with the 2D settings will attempt to generate frames with movement, as the canvas moves according to the 2D specific parameters.

Angle

Rolls the canvas clockwise or counterclockwise in degrees per frame. This parameter uses positive values to roll counterclockwise and negative values to roll clockwise.


Positive

Negative

Zoom

Scales the canvas size, multiplicatively. 1 is static, with numbers greater than 1 moving forwards and numbers less than 1 moving backwards.
